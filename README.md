# FlappyBirdAI

Flappy Bird game that uses NEAT to play. Inspired by the Tech with Tim channel (youtube.com/techwithtim).

## How to run

- Install packages using poetry:

    ```bash
    poetry install
    ```

- Run python with poetry:

    ```bash
    poetry run python flappy_bird_ai.py
    ```
